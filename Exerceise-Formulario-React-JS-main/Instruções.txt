Para usar o CRA:
    Passos iniciais n terminal:
        npx create-react-app my-app
        cd my-app
        npm start //Abre o projeto

Para usar o Vite:
    Depois limpe o terminal e rode os seguinte comandos:
        npm init vite react-vite -- --template react-ts
        cd react-vite
        npm install
        npm run dev //Abre o projeto

OBS: Use o Vite
